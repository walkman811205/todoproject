# Project_TodoList


#　介紹
所用到的技術:
<ul>
  <li>程式語言: Python、HTML/CSS</li>
  <li>框架: Django</li>
  <li>資料庫: SQLite</li>
  <li>版本控制: Git</li>
</ul>

# 功能
<ul>
 <li>目前TODO事項列表</li>
 <li>新增TODO事項</li>
 <li>標記TODO事項已完成</li>
 <li>刪除TODO事項</li>
 <li>修改TODO事項內容</li>
</ul> 


# 啟動
```
$git clone git@github.com:walkman811205/todoproject.git
$cd todoproject
```
建立虛擬環境並啟動
```
python -m venv venv
source ./venv/Scripts/activate
```
下載依賴套件
```
pip install -r requirements.txt
```
啟動django
```
python manage.py runserver
開啟瀏覽器輸入 127.0.0.1:8000 
```
